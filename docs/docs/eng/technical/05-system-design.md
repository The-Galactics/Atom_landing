# Visual Identity Guide:

## Typography:

The selection of typography aims to be the most suitable to meet our minimalist quality standards and provide a non-suffocating interaction between the user and the application.

 1. **Titles and Headings:**

 - **Font:** Geist Serif.
 - **Specification:** Used for section titles, function names, and Landing Page headings.
 - **Attribute:** Provides technical elegance and clarity, allowing for an immediate visual hierarchy without overloading the design.

 2. **Body Text and Subheadings:**

 - **Font:** Lora.
 - **Specification:** Used for all assistant responses, long descriptions, and user guides.
    
 - **Attribute:** Its design, optimized for prolonged reading, ensures that the user can interact with the AI comfortably and in a friendly manner.

# Design Principles:

- **Extreme Simplicity:** There should not be more than 3 actionable elements per view.
    
- **Negative Spacing:** White space (or black, in this case) is prioritized so that texts in **Lora** can breathe and information does not feel cluttered.
