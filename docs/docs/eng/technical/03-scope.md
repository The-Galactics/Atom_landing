# Sprint-1 Scope [Atom]: Infrastructure and Foundations.

## In-Scope:

 1. **Landing Page:** Informative site with the visual identity that leads to the App download button.
 2. **Database:** Create a basic user structure for persistence.
 3. **Audio:** The AI must be able to receive audio and respond via chat or text.
 4. **Connection:** Create the connection between Python and Java (Backend).
 5. **Documentation:** Complete technical guides and constant updates.

## Out-of-Scope:

 1. **Multilingual:** Focus primarily on understanding Spanish.
 2. **User Management:** Use only a basic profile for testing; do not implement logic.
 3. **Payment:** The MVP will focus on technical functionality, not monetization.

## Technical Restrictions:

 1. **Wi-Fi:** Internet connectivity is always required due to the consumption of external API keys.
 2. **Operating System:** Android 8.0 Oreo (API 26) or higher for library compatibility.
 3. **Minimum Hardware:** 4GB RAM or higher due to AI power requirements and background processing.
 4. **Base Language:** Java 21 and Python 3.12.
