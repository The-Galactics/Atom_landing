# Functional and Non-Functional Requirements [Atom]:

## Functional Requirements:
| ID | Category | Description |
|--|--|--|
| RF-01 | Interaction | The system must capture the user's request through text or voice and interpret this information to provide responses. |
| RF-02 | Interaction | Atom must generate responses according to the user's request, displaying text or speaking based on the assistant's configuration. |
| RF-03 | Autonomy | Atom must be capable of handling touch input and viewing the device screen to interact autonomously. |
| RF-04 | Interaction | It must listen for an activation command or trigger the floating bubble to launch the application. |
| RF-05 | Feedback | Atom must provide visual or auditory confirmation when an autonomous task has been completed, via notification or audio. |
| RF-06 | Configuration | The user must be able to adjust the AI for personalized use (Name, Nicknames, and Personality) in 3 minutes or less. |
| RF-07 | Autonomy | Atom must be able to identify the type of application (e.g., banking) to avoid violating the user's privacy and security. |
| RF-08 | Permissions | Atom must disclose the assistant's accessibility permissions through a message or voice. |
| RF-09 | Autonomy | Atom must be capable of saving the sequence of steps for a successfully executed new flow, linking it to the user's instruction for future executions. |

## Non-Functional Requirements:

| ID | Category | Description |
|--|--|--|
| RNF-01 | Interaction | The user must be able to navigate the application interface easily, with a maximum of 3 buttons to reach or configure important settings. |
| RNF-02 | UI | It must follow minimalist design principles to reduce cognitive load and have integrated assistive technology. |
| RNF-03 | Availability | The application must function correctly with a minimum 3G Internet connection. |
| RNF-04 | Performance | The response time between the request and the action must not exceed 4 seconds. |
| RNF-05 | Security | The application must encrypt passwords to protect privacy (AES-256). |
