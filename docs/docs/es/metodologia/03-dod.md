# Definition of Done (DoD) - [Atom]:

Este documento establece los criterios mínimos que cada Historia de Usuario (HU) debe cumplir para ser considerada como "Terminada" dentro de un Sprint.

## Criterios de Calidad Generales- Sprint 1:

- **Código Limpio:** El código sigue los estándares del equipo, no tiene errores de compilación y se ha eliminado código comentado o innecesario.
- **Arquitectura:** Se respeta la Arquitectura Hexagonal y la separación de capas (Domain, Infra, IA). 
- **Pruebas:** Se han ejecutado y aprobado las pruebas unitarias correspondientes. 
- **Seguridad:** Los datos sensibles están protegidos con cifrado AES-256. 
- **Accesibilidad:** La interfaz cumple con los requisitos de baja carga cognitiva y es apta para el nicho de discapacidad visual/motriz. 
- **Documentación:** Se ha actualizado el manual técnico en Docusaurus y se han comentado los métodos complejos de la App. 
- **Revisión:** El Product Owner ha validado que la funcionalidad cumple con los Criterios de Aceptación.
