# Objetivos Del Proyecto [Atom].

## Objetivo General:

Desarrollar un sistema multi-agent, basado en AI generativa para crear respuestas en tiempo real, con el fin de centralizar las tareas del usuario en un solo punto, automatizando el flujo manual para acceder o manejar el dispositivo, por medio de comando de voz o texto interpretando lenguaje natural.
