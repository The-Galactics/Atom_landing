# Scope Sprint-1 [Atom]: Infrastructura y Bases.

## In-Scope:

 1. **Landing Page:** Sitio informativo con la identidad visual que lleve hasta el botón de descarga de la App.
 2. **Database:** Crear estructura basica de usuarios para persistencia
 3. **Audio:** La AI debe ser capaz de recibir audio y responder por chat o textos.
 4. **Connection:** Crear la conexión de python con java (Backend)
 5. **Documentation:** Guias tecnicas completas y constante actualización.

## Out-Scope:

 1. **Multilingue:** Enfocarse principalmente en que entienda español.
 2. **User Management:** Utilizar solo un perfil básico para pruebas, no implementar lógica.
 3. **Payment:** El MVP se centrara en la funcionalidad técnica, no en la monetización.

## Technical Restrictions:

 1. **WI-FI :** Siempre debe haber conectividad a internet por consumo de API keys externas.
 2. **Operative System:** Android 10 o superior por compatibilidad de librerias.
 3. **Minimum Hardware:** 4GB o superior por la potencia de la AI y el segundo plano.
4. **Base Language:** Java 21 and Python 3.13.
