# Requisitos funcionales y no funcionales [Atom]:


## Requisitos Funcionales:
|ID  | Category | Descripción |
|--|--|--|
|RF-01  |Interacción  | El sistema debe capturar la petición del usuario por medio de texto o voz y debe interpretar esta información para proporcionar respuestas .|
|RF-02  |Interacción  | Atom debe generar respuestas de acuerdo a la petición del usuario, mostrando texto o hablando según la configuración del asistente.|
|RF-03 | Autonomía | Atom debe ser capaz de manejar el touch y ver la pantalla del dispositivo para interactuar de manera autónoma. |
|RF-04| Interacción | Debe escuchar un comando de activación o accionar la burbuja flotante para iniciar la aplicación.|
|RF-05| Retroalimentación | Atom debe confirmar visual o auditivamente cuando una tarea autónoma ha sido completada, por medio de una notificación o audio.|
|RF-06 | Configuración | El usuario debe poder ajustar la AI para uso personalizado (Nombre, Apodos y Personalidad) en 3 minutos o menos.|
|RF-07 | Autonomía | Atom debe ser capaz de identificar el tipo de aplicación (ej. bancaria) para no violar la privacidad y seguridad del usuario.|
|RF-08| Permisos | Atom debe dar a conocer los permisos de accesibilidad del asistente por medio de un mensaje o voz|
|RF-09| Autonomia | Atom debe ser capaz de guardar la secuencia de pasos de un flujo nuevo ejecutado con éxito, vinculándolo a la instrucción del usuario para futuras ejecuciones.|

## Requisitos No Funcionales:

|ID| Categoría |Descripción|
|--|--|--|
|RNF-01 | Interacción | El usuario debe poder navegar de manera sencilla por la interfaz de la aplicación, máximo 3 botones para llegar o configurar ajustes importantes.|
|RNF-02 | UI | Debe seguir principios de diseño minimalista para reducir la carga cognitiva. y tener integrada tecnología asistida|
|RNF-03| Disponibilidad | La aplicación debe funcionar correctamente con una conexión a Internet mínima 3G.|
|RNF-04 | Rendimiento |El tiempo de respuesta entre la petición y la acción no debe superar los 4 segundos.|
|RNF-05 | Seguridad |La aplicación debe cifrar contraseñas para proteger la privacidad (AES-256)|
